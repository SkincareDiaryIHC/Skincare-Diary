CONSULTORA


DE BELLEZA

Te ayudará a encontrar a las mejores consultoras de belleza de confianza, que te recomendará productos de calidad y te ayudarán a cuidar mejor tu piel

EN LA APLICACIÓN PODRÁ REALIZAR LA BÚQUEDA MEDIANTE:

FILTRO POR MARCA

Para una mejor búsqueda podrá realizar un filtro para encontrar a las consultoras según para la marca para la que trabajen.

FILTRO POR HORARIO

Para una mejor búsqueda podrá realizar un filtro para buscar a la consultora, según el horario que tengan disponible para una asesoría.

COMO CONSULTORA DE BELLEZA PODRÁ ACCEDER:

ASESORÍAS

Podrá visualizar la fecha y hora de las asesorías que reserven los distintos clientes, así como también ver el perfil de cliente para poder resetarle mejor algún producto.

HORARIO

Podrá editar y guardar su horario, según las fechas en la que se encuentre disponible, de esta forma será accesible para los clientes para que elijan alguna asesoría con usted.
